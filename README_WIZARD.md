## Ʊֆǟɢɛ

### 1. Ɨռȶɛʀǟƈȶɨʋɛ Ɱօɖɛ

Ֆɨʍքʟʏ ʀʊռ ȶɦɛ ƈօʍʍǟռɖ աɨȶɦօʊȶ ǟʀɢʊʍɛռȶֆ ȶօ ɛռȶɛʀ ɨռȶɛʀǟƈȶɨʋɛ ʍօɖɛ:

```bash

```

Ɨռȶɛʀǟƈȶɨʋɛ ʍօɖɛ ʄɛǟȶʊʀɛֆ:
- **Ǟʊȶօ-ɖɛȶɛƈȶɨօռ**: Ǟʊȶօʍǟȶɨƈǟʟʟʏ ɖɛȶɛƈȶֆ ɨʄ ɨռքʊȶ ɨֆ ռօʀʍǟʟ ȶɛӼȶ օʀ աɨʐǟʀɖ ƈɦǟʀǟƈȶɛʀֆ
  - Ռօʀʍǟʟ ȶɛӼȶ (a-z, A-Z) → Ƈօռʋɛʀȶֆ ȶօ աɨʐǟʀɖ ƈɦǟʀǟƈȶɛʀֆ
  - Ɯizard ƈɦǟʀǟƈȶɛʀֆ → Ƈօռʋɛʀȶֆ ɮǟƈӄ ȶօ ռօʀʍǟʟ ȶɛӼȶ
- **Ֆɨʍքʟɛ**: Ռօ ƈօʍʍǟռɖֆ ռɛɛɖɛɖ, ʝʊֆȶ ȶʏքɛ ǟռɖ ƈօռʋɛʀȶ
- `Ctrl+D` - ƐӼɨȶ

    ║   Ƭʀǟռֆʄօʀʍ ȶɛӼȶ ɨռȶօ ʍǟɢɨƈǟʟ ƈɦǟʀǟƈȶɛʀֆ     ║
    ╚══════════════════════════════════════════════╝

Ƭʏքɛ ȶɛӼȶ ȶօ ƈօռʋɛʀȶ (ǟʊȶօ-ɖɛȶɛƈȶֆ ɖɨʀɛƈȶɨօռ), Ƈȶʀʟ+Ɖ ȶօ ɛӼɨȶ
